Introduction

JavaScript es un lenguaje de programación de alto nivel y de propósito general. Utiliza un enfoque multiparadigma, lo que significa que admite la programación procedural, orientada a objetos y algunos conceptos de programación funcional.

Fue creado originalmente por Brendan Eich en un período de 10 días en 1995 como un lenguaje de scripting para mejorar la interactividad en páginas web en el navegador Netscape Navigator. Desde entonces,ha evolucionado enormemente y se ha convertido en uno de los lenguajes de programación más populares y ampliamente utilizados en la web.

Operadores

Los operadores en JavaScript son símbolos o palabras reservadas que se utilizan para realizar operaciones en variables y valores.

  • Equals: numero1 == numero3
  • Not Equals: numero1 != numero3
  • Less than: numero3 < numero1
  • Less than or equal to: numero3 =< numero1
  • Greater than: numero1 > numero3
  • Greater than or equal to: numero1 >= numero3

Estas condiciones se pueden utilizar de varias maneras, más comúnmente en "declaraciones if" y bucles.

const numero1 = 20; const numero2 = "20"; const numero3 = 30; // Operador Mayor a console.log( numero1 > numero3 ); console.log( numero3 > numero1 ); // Operador Menor que console.log(numero3 < numero1);

Functions

En JavaScript, una función es un bloque de código que realiza una tarea específica o calcula un valor. Las funciones en JavaScript son objetos de primera clase, lo que significa que pueden ser tratadas como cualquier otra variable, lo que permite pasarse como argumentos a otras funciones, asignarse a variables y devolverse como valores desde otras funciones.

Funcion sin parametros

var a=10, b= 20 function greetings(){ return "Hello, Friend"; } document.querySelector('#funcion-1').innerHTML = greetings()

Funcion con parametros

var a=10, b= 20 function suma(numero1, numero2){ return numero1 +numero2; } document.querySelector('#funcion-2').innerHTML = suma(a, b)

Objetos

En JavaScript, un objeto es una estructura de datos que puede contener propiedades y métodos. Las propiedades son pares clave-valor que almacenan datos, mientras que los métodos son funciones asociadas al objeto que pueden realizar acciones o cálculos específicos.

const articulos = { nombre:'cama', size: 'grande', color:'blanca', precio:'$35.00', peso: '60kg', modelo: 'RD0144' }

Arrays

En JavaScript, un array es una estructura de datos que se utiliza para almacenar una colección ordenada de elementos. Estos elementos pueden ser de cualquier tipo de datos, como números, cadenas, objetos, funciones, etc.

var colores = ['rojo', 'verde', 'azul'];
